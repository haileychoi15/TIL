# 자료형(data type)
최신 ECMAScript 표준은 9개의 자료형을 정의합니다.

### Primitive types
- `Boolean`
- `undefined`
- `Number`
- `String`
- `BigInt`
- `Symbol` : 유일하고 변경 불가능한 (immutable) 기본값

### Special primitive type
- `null`

### Structural Types
- `Object`
- `Function`

<br /><br />

## Falsy values
Boolean context에서 `false`로 평가되는 자료형들 입니다.
- `undefined`
- `null`
- `''`
- `0`
- `NaN`

> 위의 값들을 제외한 모든 값은 Truthy value 입니다.
